Latest Project

Gambler's Paradise

Gambler's Paradise is an entertainment application that allows users to create an account and gamble with in game currency. The application features five different games and a login authetication system. It was built using the MERN stack and was styled with Bootstrap.

Other Projects

Daily Weather App

This weather app= allows users to enter in their city and will show the weather forecast in the location for the next week. It was built with Vanilla JavaScript, HTML, and CSS.

Shopping Cart Application

This shopping cart application allows users to place items into a cart and then it gives them a subtotal for their items. It was built using React and Bootstrap.

Resume Builder

This resume builder allows for users to input the information that a resume would require and then when the user submits their information, it will place their information in a resume format. It was built with React and Bootstrap.

Mascot Mania

Mascot Mania is a game that allows users to test their memories and keeps track of their current wins and how many cards they have gotten in a row without clicking on the same card twice. It was built using React and Bootstrap.
